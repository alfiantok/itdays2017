    Mural

    Mural berasal dari kata ‘murus’, sebuah kata dari Bahasa Latin yang memiliki arti dinding. Mural adalah lukisan berukuran besar yang dibuat pada dinding (interior ataupun eksterior), langit-langit, atau bidang datar lainnya.

    Mural tidak hanya berbentuk tulisan saja. Mural sifatnya lebih bebas. Biasanya karya-karya seni mural lebih sering berbentuk gambar. Namun, kadang gambarnya berkonsepkan dengan isu sosial masa kini. Kadang kala dalam satu bentuk mural terdapat satu tokoh inti cerita yang kadang tidak berbentuk orang. Namun, memiliki arti yang bisa membuat semua orang yang melihat, langsung mengetahui apa arti dibalik gambar tersebut.

    Mural dipergunakan sebagai ajang kegiatan spiritual dan ajang eksistensi diri. Kini mural bukan hanya sebagai media pelengkap interior, meskipun fungsi tersebut tidak benar benar hilang namun mengalami perkembangan fungsi, lokasi, teknik, media dan thematic mural makin meluas sekaligus menguat dikalangan artist maupun street artist baik sebagai media propaganda sosial, kampanye instansi, idealisme dan ekspresi seniman itu sendiri. Mural kini lebih dikenal sebagai seni publik karena lokasi keberadaanya, tema maupun penyampaiannya yang melibatkan langsung interaksi maupun opini masyarakat, membuat mural menjadi seni visual yang merakyat baik dan buruk semua itu dikembalikan pada interaksi dan pertanggung jawaban sang seniman dan masyarakat

    Lomba Mural ini diusung Panitia IT DAYS 8 sebagai salah satu perlombaan dalam rangkaian acara IT DAYS. Dalam kompetisi ini peserta diajak turut serta menuangkan ide kreatifnya untuk memberikan kreatifitasnya melalui karya seninya. Lomba Mural sendiri bertemakan “Seni Mengispirasi Teknologi”.

    Web design

    World Wide Web (WWW) adalah salah satu bagian dari internet. WWW sendiri disebut juga sebagai Web, adalah sebuah sistem interlink dokumen hyperteks yang diakses melalui internet. Web dapat diakses menggunakan Web Browser. Dengan menggunakan Web Browser, pengguna internet dapat melihat halaman suatu web yang dapat berisi teks, gambar, video atau multimedia lain yang dinavigasi menggunakan hyperlink.

    Web sendiri dibedakan menjadi dua jenis, yaitu Web Statis dan Web Dinamis. Web Statis adalah sebuah halaman web yang berisi informasi yang ditulis dengan HTML. Informasi pada halaman web statis tidak dapat diubah kecuali dengan mengubah source code HTML yang membentuk halaman tersebut. Sedangkan Web Dinamis adalah halaman yang di-generate pada setiap akses yang diminta oleh user atau halaman web yang selalu berubah sebagai hasil dari interaksi dengan user. Halaman web dinamis umumnya berupa template untuk menampilkan hasil dari operasi kueri terhadap sumber informasi, misalnya database atau hasil kalkulasi dari suatu algoritma program.

    Web juga dibedakan kembali antara bagian Client dan Server. Pada bagian Client source code aplikasi web dijalankan oleh web browser ataupun plug-in browser. Aplikasi web ini diunduh dari server web dan diproses di client web, contohnya adalah JavaScript dan ActionScript. Pada bagian Server source code aplikasi web dijalankan di web server untuk membangkitkan web dinamis. Hasil dari pemrosessan informasi di server dikirimkan ke web browser dalam bentuk dokumen web, contohnya JSP, Apache dan PHP.

    Web Design Competition diusung Panitia IT DAYS 8 sebagai salah satu perlombaan dalam rangkaian acara IT DAYS. Web Design sendiri termasuk pada Web Development. Web Development merupakan pembangunan dan pengembangan web site untuk WWW pada jaringan internet maupun intranet. Selain Web Design, adapula Web Content Development, Client/Server-side Scripting, dan Konfigurasi Web Server. Jangkauan definisi dari pembuatan halaman web sendiri terhitung dari Web Statis tunggal sampai Web Dinamis yang kompleks, contohnya Aplikasi Internet, Electronic Businesses, atau SNS (Sosial Networks Service biasa dikenal sebagai Media Sosial).

    Dalam kompetisi ini peserta diajak turut serta menuangkan ide kreatifnya untuk memberikan kreatifitasnya melalui desain web. Diusungnya “Art of Technology” sebagai tema utama IT DAYS 8, mengajak para peserta Web Design Competition untuk membuat design web yang bisa menggabungkan seni dan teknologi melebur menjadi satu.

    Kompetisi ini ditunjukkan untuk seluruh siswa SMA/SMK/MA sederajat di wilayah D.I.Y. & Jawa Tengah. Kompetisi ini diikuti dalam tim dengan anggota maksimum 3 orang. Pelaksaannya akan dibagi menjadi 2 babak, yaitu babak penyisihan dan babak final. 10 Tim terbaik akan diundang untuk mempresentasikan karyanya di hadapan juri di Universitas Sanata Dharma.

    Lomba Internal

    Lomba Internal ini ditujukan khususnya pada mahasiswa Universitas Sanata Dharma.

    Lomba terdiri dari :

    - Fun Game

    - Futsal

    - Searching

    - Basket

    ( FUN GAME ) Peserta dengan tim maksimal 4 orang, akan memakai bando dan karung kemudian akan berperang dengan tim lawan menggunakan pistol air. Lomba ini diadakan di Hall Selatan Kampus III Universitas Sanata Dharma. Lomba ini diadakan pada hari Jumat, 29 Agustus 2017.

    ( FUTSAL ) Lomba akan diadakan pada hari Senin, 8 September 2017 - Rabu, 15 September 2017 dan babak final akan dilaksanakan pada hari Jumat, 29 September 2017. Lomba ini akan diikuti 10 tim futsal laki-laki (tiap angkatan dari 2013, 2014, 2015, 2016, 2017 maksimal 2 tim) dan 4 sampai 10 tim futsal perempuan. Lomba akan dimulai pukul 18.00 sampai selesai di lapangan futsal kampus paingan. Lomba ini akan diambil 2 pemenang untuk tim laki - laki dan 2 pemenang untuk tim perempuan.

    ( SEARCHING ) Lomba akan diadakan pada hari Jumat, 29 September 2017 pukul 09.00 sampai selesai di Hall Selatan Kampus 3 USD. Peserta terdiri dari mahasiswa teknik informatika. Peserta akan diberikan soal oleh panitia, kemudian peserta akan mencari jawaban di internet. Peserta diberi waktu 1 menit untuk melakukan searching kemudian peserta akan menuliskan jawaban pada sebuah kertas dan mengangkat jawaban diatas kepala. Bagi peserta yang jawabannya benar akan melanjutkan ke tahap soal berikutnya, peserta yang jawabannya salah akan meninggalkan tempat lomba. Target peserta lomba ini adalah 50 orang yang kemudian akan diambil 1 orang pemenang.

    ( BASKET ) Lomba akan diadakan pada hari Senin, 8 September 2017 - Rabu, 15 September 2017 dan babak final akan dilaksanakan pada hari Jumat, 29 September 2017. Lomba ini akan diikuti 10 tim basket laki-laki (tiap angkatan dari 2013, 2014, 2015, 2016, 2017 maksimal 2 tim) dan 4 sampai 10 tim basket perempuan. Tiap tim terdiri dari maksimal 10 pemain dan minimal 6 pemain. Lomba akan dimulai pukul 18.00 sampai selesai di lapangan basket kampus paingan. Lomba ini akan diambil 2 pemenang untuk tim laki-laki dan 2 pemenang perempuan.
